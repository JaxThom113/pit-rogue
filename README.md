# Pit Rogue

Roguelike game where you have to climb 10 levels to escape the Pits of Doom!

Use the following commands to set up venv and run the game:

```
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

The project is based on the RoguelikeDev 2020 Tutorial Series and is forked from the reference repository: https://github.com/TStand90/tcod_tutorial_v2
